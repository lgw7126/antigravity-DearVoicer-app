1. 제작 동기 및 기획 목적

  • 발화 보조 수단 제공: 목소리를 내어 대화하는 데 일시적 또는 만성적인 어려움이 있으신 분들이 쉽고 빠르게 대화를 나눌 수 있도록 돕고자 제작되었습니다.
  • 나만의 개성 있는 음성 구현: 기존의 기계적이고 차가운 안내음 대신, ElevenLabs의 첨단 기술을 활용하여 감성적이고 자연스럽게 감정이 살아있는 음성을 출력합니다.
  • 편의성과 미려함: 조작이 서투르신 분들도 직관적으로 이해할 수 있는 큼직한 버튼과 깔끔하게 배치된 슬라이더, 눈의 피로를 낮추어주는 다크 모드 등을 결합한 프리미엄 UI를 갖추었습니다.

2. 구현 과정

  1. 안전한 보안 설계: ElevenLabs API Key가 소스코드에 하드코딩될 경우 유출 위험이 커지므로, 앱 화면 내 우측 상단 설정창(⚙️)을 통해 사용자가 직접 본인의 Key를 입력하고, 이를 브라우저의 로컬 보안 영역(localStorage)에만 안전하게 보관하여 유출 걱정 없이 사용 가능하도록 구축했습니다.
  2. 사용 편의성 강화:
    • 자주 사용하는 문구를 추가/삭제할 수 있는 동적 리스트(CRUD) 구현.
    • 속도 조절 슬라이더 외에 퀵 프리셋 버튼(0.8x, 1.0x, 1.2x) 및 미세 조절 버튼(+/-) 연동.
    • [시너지] 감정 톤 프리셋: 다정하게(부드러움), 차분하게(기본형), 또박또박(정확성) 등 원하는 감정의 칩을 선택해 ElevenLabs API의 stability와 similarity_boost 값을 다르게 제어하도록 구현.
    • [시너지] 초성 매칭 문장 추천: 입력창 타이핑 시 자동으로 연관 문장을 오버레이 말풍선으로 추천. 특히 초성 검색(예: ㅁㅇ -> 미안해 고마워)을 지원하여 타이핑을 최소화하도록 편의성 증대.
  3. 재생 및 대기 상태 제어: 오디오 재생 도중 언제든지 정지할 수 있는 '멈추기' 기능과 AI 음성 변환 중임을 알려주는 스피너 애니메이션 처리 완료.

📢 대화하고 소리 내기

  • 말하기: 본문창에 말할 내용을 적고 주황색 [말하기] 버튼을 누릅니다.
  • 멈추기: 오디오가 재생 중일 때 버튼이 빨갛게 [🛑 멈추기]로 변합니다. 클릭 시 즉시 재생이 차단됩니다.
  • 지우기: 입력창의 내용을 한번에 지우고 재생 중이던 소리를 정지하려면 [지우기]를 누릅니다.
  • 감정 톤 조절: 입력창 하단의 감정 칩(😊다정하게, 😌차분하게, 📢또박또박)을 선택하면 해당 감정 톤으로 음성이 출력됩니다. (재생 도중에 누르면 즉시 멈추고 톤이 변경됩니다.)
  • 스마트 추천 단어 완성: 검색창에 한글을 치거나 초성만 입력하면(예: `ㅁㅇ` 또는 `미안`) 하단에 추천 말풍선이 뜹니다. 이 말풍선 칩을 터치하면 문장이 자동으로 완성됩니다.
  • 자주 하는 말: 아래 준비된 8개의 퀵 카드를 누르면 즉시 해당 대사가 읽혀 나갑니다.
  • 커스텀 문구 추가: [+ 문구 추가] 버튼을 눌러 나만의 자주 쓰는 대사와 이모지를 추가하거나, 카드 우측 상단의 작은 X 아이콘을 눌러 지울 수 있습니다.
  • 카카오톡 전송: [💬 카톡 전송] 버튼을 누르면 입력창의 텍스트가 카카오톡 친구나 채팅방으로 즉시 공유됩니다.
